# Contour Agent Guide

Contour is a multi-skill repository for graspable visual inquiry.

When working in this repository, prefer the `skills/` layer for task-specific guidance.
Use `docs/` as the design/specification reference layer, and `examples/` as the artifact corpus.

## Core stance

Contour is not a generic diagramming tool.
It is a cognitive instrument for graspable inquiry.

Optimize for:
- orientation
- revisability
- legibility
- visual calm
- conversational usefulness

Avoid:
- dashboard energy
- generic flowchart defaults
- product chrome
- decorative excess

## Skill selection

Choose the most relevant skill before acting:
- scene creation or editing → `skills/contour-scene-authoring/SKILL.md`
- rendering or export work → `skills/contour-rendering/SKILL.md`
- checking output quality/constraints → `skills/contour-svg-validation/SKILL.md`
- refining visual language or aesthetic direction → `skills/contour-style-evolution/SKILL.md`

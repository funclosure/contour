# contour-scene-authoring

Use this skill when turning a conversation, concept cluster, or inquiry shape into a Contour scene representation.

## Purpose

Translate thought structure into a small, revisable scene artifact.

## Focus

Prefer:
- distinctions
- tensions
- relationships
- clusters
- landmarks
- paths of inquiry

Avoid prematurely over-specifying layout details when the structure is still unclear.

## Inputs

Typical inputs include:
- chat summaries
- concept lists
- rough outlines
- existing `.scene.json` files

## Outputs

Produce or revise:
- `.scene.json` artifacts
- scene-ready structural drafts

## Working rules

- Optimize for revisability.
- Keep the representational structure small and editable.
- Preserve ambiguity where it supports inquiry.
- Name important elements clearly enough to act as landmarks.

# contour-rendering

Use this skill when transforming Contour representations into visible artifacts for review or delivery.

## Purpose

Render a scene into SVG and, when needed, chat-safe delivery formats.

## Outputs

Typical outputs:
- `.svg`
- `.png`
- optional `.html` wrappers

## Working rules

- Prefer SVG as the canonical visual artifact.
- Treat HTML as optional presentation, not the core carrier.
- Use PNG when conversational delivery requires a raster format.
- Preserve legibility over visual flourish.

## Delivery stance

Contour is incomplete if the artifact cannot re-enter the conversation and shape the next move.

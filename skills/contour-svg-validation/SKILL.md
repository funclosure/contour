# contour-svg-validation

Use this skill when checking whether a Contour output is structurally valid, visually legible, and safe for delivery.

## Purpose

Validate generated SVG and related artifacts against Contour's rendering and communication constraints.

## Check for

- readable text
- sensible hierarchy
- clear grouping
- non-broken layout
- export readiness
- conversational usability

## Working rules

- Legibility is sacred.
- Validation is not only technical, it is cognitive.
- A visually correct artifact can still fail if it does not improve orientation.

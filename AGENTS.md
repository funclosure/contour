# Contour Agent Notes

Contour is organized as a reusable skill library for representation workflows.

Default habit:
1. choose the relevant skill in `skills/`
2. use `docs/` for theory/spec detail
3. use `examples/` for concrete patterns and test artifacts

Preserve the distinction between:
- **skills**: agent instructions and reusable workflows
- **docs**: architecture, model, and visual design reference
- **examples**: scenes and rendered outputs
